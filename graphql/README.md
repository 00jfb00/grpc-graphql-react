# GraphQL

## Overview

This project has the following directory structure:

```
.
└── graphql
    └── src
        └── app
            ├── resolvers
            ├── schema
            └── services
```

### 1. Install dependencies

```sh
npm install
```

### 2. Running the project

```sh
npm start
```

The project will run on http://localhost:4000/graphql

### 3. Building the project

```sh
npm run build
```
