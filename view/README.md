# View

## Overview

This project has the following directory structure:

```
.
└── view
    ├── public
    └── src
        ├── assets
        └── constants
        └── pages
```

### 1. Install dependencies

```sh
npm install
```

### 2. Running the project

```sh
npm start
```

The project will run on http://localhost:3000

### 3. Building the project

```sh
npm run build
```
